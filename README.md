# This is a sample Microservices Project

> Use full commands :

- For development

```bash
skaffold dev
```

- For production

```bash
skaffold run
```

- For delete all resources

```bash
skaffold delete
```

> Edit host file :

- For MacOS

```bash
/etc/hosts
```

- For Windows

```bash
C:\Windows\System32\Drivers\etc\hosts
```
